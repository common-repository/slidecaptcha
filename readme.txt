Slide Captcha  it is a WordPress plugin that   add a slide captcha field to the WordPress comment form.  

== Description ==
This captcha does not use any jquery plugins or libraries, only JS and jQuery, it means it has very light weight, also it use simple checking to define is  real man  leaved the comment  or bot
[Live Demo](http://captcha.web-andryshka.ru/index.php/2017/06/12/hello-world/ "Live Demo") 
== Installation ==

1. Install Slider CAPTCHA either via the WordPress.org plugin directory, or by uploading the files to your server (`/wp-content/plugins/`) 
2. Activation can be made in 'Plugins' menu
3. Configure disign in the settings of plugin.
4. Have fun.
